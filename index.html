How does javascript work?

The javascript code we write in any platform, the code passes through compiler. The compiler translates it into bytecode which is also known as machine code.Javascript has no compilation process.An interpreter in the browser reads the code and interprets line by line and executes it.

what does Javascript event loop do?

Javascript is a asynchronous and single-threaded programming language.When an async function is called, it is sent to a browser API.API starts its own single-threaded operation like setTimeout. it is sent to the corresponding API like stack which waits till the specified time to send this operation back for processing.it sent to the event queue.The event loop runs this process; it continuously, checks whether the call stack is empty or not. If it is empty, new functions add from the event queue. If it is not, then the current function call is executed.
